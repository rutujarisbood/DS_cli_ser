Open server.sln in visual studio

Keep the setting on multiple startup projects(already exists)
First run backup server and then primary server.
On server: click 'start' to start the backup server
run another instance of server for primary server
once server is started
enter client name and click 'start'
once connection is accepted 'upload file'
	-the default path is set to "C:\sample files\serverFile.txt.txt" for lexicon file (primary server)
		and "C:\sample files\serverFile2.txt.txt" for lexicon file (backup server)
	-please copy "sample files' folder to C:/
once file is uploaded server will process the file and send the output
enter word to be added to lexicon through GUI

on 'exit' click server/client will be closed
